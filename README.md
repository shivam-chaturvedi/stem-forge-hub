# STEM Blog + Project Showcase 🚀

A platform to **write technical blogs**, **showcase innovative projects**, and explore cutting-edge STEM topics.  
It connects developers, researchers, and tech enthusiasts by enabling knowledge sharing and collaboration.

---

## 🌟 Features

- ✍️ **Write Blog Posts** about your learning journey (e.g., *How I Built My First Python Game*).
- 📂 **Add Projects** with GitHub links, screenshots, and detailed explanations.
- 🏷 **Tagging & Categorization System**: Organize blogs and projects into fields like IoT, Web Dev, AI, Robotics, and more.
- 🌙 **Dark Mode Toggle** for a better reading experience.

## 🚀 Tech Stack

- **Frontend**: React.js, Tailwind CSS  
- **Backend**: Django / Spring Boot (optional for project linking)  
- **Database**: PostgreSQL / MySQL  
- **Other Tools**: Redis, WebSockets, IoT integrations (ESP32, AWS, etc.)

---

## 📌 How to Run

1. Clone this repository  
   ```bash
   git clone https://github.com/your-username/stem-blog-showcase.git
   cd stem-blog-showcase
   ```

2. Install dependencies  
   ```bash
   npm install   # for frontend
   ```

3. Start the development server  
   ```bash
   npm run dev
   ```

4. Open in browser: [[http://localhost:8080](http://localhost:8080)](https://preview--stem-forge-hub.lovable.app/)
